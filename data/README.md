# Letter Data

This folder includes extracted Markdown copies of Jeff Bezos's Amazon shareholder letters from 1997 through 2020.

The Markdown files are derived from official Amazon-hosted PDF sources listed in `SOURCES.md`. Each extracted file also includes the author and source URL in its header.

The raw PDFs, SQLite database, evaluation reports, and vector indexes are generated locally and are not committed:

- `data/raw/`
- `data/db/`
- `data/evals/`
- `data/indexes/`
